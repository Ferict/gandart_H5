# 提交清单

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-17
> 说明：本目录用于承载复杂提交边界或用户明确要求时的提交范围清单，不作为默认必写目录，也不作为规则真值源或历史主档案目录。

## 1. 目录职责

本目录用于记录单次或少量连续批次的提交边界说明，帮助人工复核以下问题：

1. 本批提交包含哪些文件或主题。
2. 本批提交明确排除了哪些无关改动。
3. 本批提交与当前计划、审查、台账的对应关系是什么。

## 2. 当前使用规则

当前口径固定为：

1. 只有在提交边界复杂、存在并行脏改，或用户明确要求时，才启用提交清单。
2. 简单批次不默认新增提交清单文件。
3. 本目录内容只作为提交流程辅助材料，不作为现行规则、接口契约或历史总账真值源。

## 3. 历史常见内容与后续新增方式

历史上曾出现的文件包括：

1. 冻结旧长流水：`当前提交清单.md`

当前路径不再保留 `当前提交清单.md`。后续如需重新启用提交清单，应新增新的按日或按批次提交边界文件，而不是续写旧长流水。

## 4. 不应放入的内容

以下内容不应写入本目录：

1. 现行规则正文
2. 正式接口契约或联调说明
3. 长期历史记录总账
4. 与提交边界无关的过程性噪音记录
