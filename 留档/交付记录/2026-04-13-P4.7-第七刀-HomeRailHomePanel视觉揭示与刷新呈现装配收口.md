## 2. 收口结果

1. `useHomeRailHomePanelRuntime` 不再直接持有以下装配顺序：
   - `useHomeHomeVisualReveal(...)`
   - `useHomePagePresentationRuntime(...)`
2. 新 `useHomeRailHomeVisualPresentationRuntime` 统一承接：
   - banner / featured / market 三类视觉揭示
   - 顶部 refresh presentation replay
   - pull-refresh 期间的本地 presentation wait
   - scene patch motion reduction 与公告 live reorder runtime
3. `useHomeRailHomePanelRuntime` 继续保留：
   - market query 与 remote list pipeline
   - result window
   - scene patch
   - content lifecycle
   - presentation state
   - effects

## 3. 风险控制

1. 本刀只搬装配层，不修改 `useHomeHomeVisualReveal` 与 `useHomePagePresentationRuntime` 的对外接口。
2. 新 helper 只暴露 `HomeRailHomePanel` 需要的现成字段，不把内部实现细节继续扩散到父层。
3. `reloadMarketList(...)` 的 refresh replay 行为仍由原先回调驱动，未改变触发条件与 motion source。
