# 批次377-首页卡片双层阴影与压暗复查

## 目标

- 基于最新截图，复核首页市场卡当前可见的视觉问题是否同时包含静态阴影与长按压暗层。

## 审查范围

- `src/pages/home/components/HomeRailHomePanel.vue`
- `src/pages/home/components/HomeInteractiveTarget.vue`

## 审查结果

1. 首页市场卡当前同时存在两层视觉反馈。

- 静态卡面阴影：
  - `HomeRailHomePanel.vue`
  - `.home-market-card-visual { box-shadow: var(--aether-shadow-soft, 0 0 24px rgba(15, 23, 42, 0.06)); }`
- 长按压暗层：
  - `HomeInteractiveTarget.vue`
  - `.home-interactive-target.is-pressed .home-interactive-target-hit { background: rgba(15, 23, 42, 0.04); }`

2. 用户截图证明静态卡面阴影仍然可见。

- 截图是在非长按状态下也能看见卡面底部和边缘的柔阴影。
- 这条不能再归因到按压态。

3. 当前实现与用户口径不一致。

- 用户当前口径至少包含：
  - 不要长按时那层半透明压暗底色
  - 市场卡现阶段也不接受这种显性的卡面阴影感

## 结论

- 不是“二选一”的问题。
- 当前首页市场卡同时保留了：
  1. 静态 `box-shadow`
  2. 长按压暗背景
- 若要完全收口，下一批应同时删除这两层，而不是只删其中一条。
