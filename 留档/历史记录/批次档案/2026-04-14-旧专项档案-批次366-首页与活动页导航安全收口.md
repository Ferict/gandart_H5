# 批次366-首页与活动页导航安全收口

## 目标

- 将首页主面板与活动页的页面跳转统一接入 `navigateByUrlSafely()`。
- 本批不处理个人中心页面跳转，按用户口径后置。

## 修改范围

- `src/pages/home/components/HomeRailHomePanel.vue`
- `src/pages/home/components/HomeRailActivityPanel.vue`

## 实际修改

### 首页主面板

- 删除本地 `safeNavigate => uni.navigateTo` 包装。
- 以下点击入口统一改走 `navigateByUrlSafely()`：
  - banner
  - 公告栏
  - 首发藏品卡
  - 市场搜索
  - 市场文档
  - 市场卡片

### 活动页

- 以下点击入口统一改走 `navigateByUrlSafely()`：
  - 活动入口卡
  - 公告卡片

## 明确不改

- `HomeRailProfilePanel.vue`
- `navigateByUrlSafely()` helper 本身
- 其它正式页

## 验证

- `npm run type-check`
- `npm run build:h5`
- `npm run build:app`
- `npm run build:app-harmony`
