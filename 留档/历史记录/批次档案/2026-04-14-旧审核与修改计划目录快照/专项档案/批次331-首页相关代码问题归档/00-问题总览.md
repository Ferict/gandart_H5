# 批次331-首页相关代码问题归档

对应审查：

- 审核记录 330：首页相关代码复查
- AI审查记录 024：首页相关代码复查

归档目标：

- 单独保留本轮首页审查命中的问题代码与关联代码。
- 便于后续按问题逐项修复，不再回头翻整份审查台账。

本批次确认的问题：

1. `HomeShellTrackStage.vue`

- 问题：仍保留 `>=768` 自动切到 `mode-web` 的独立壳层分支。
- 影响：首页当前实际维护两套壳层路径，后续逐页校对若只修移动主路径，宽屏分支会继续失真。
- 相关摘录：`01-HomeShellTrackStage-双壳层分支.md`

2. `HomeShellDrawer.vue`

- 问题：共享抽屉跳转使用 `window.setTimeout`。
- 影响：组件被直接绑到浏览器运行时，不符合 uni-app 共享壳层边界。
- 相关摘录：`02-HomeShellDrawer-window-setTimeout.md`

3. `HomeRailActivityPanel.vue`

- 问题：内容链加载缺少 `try/catch`。
- 影响：内容服务失败时会出现未处理 rejection。
- 相关摘录：`03-HomeRailActivityPanel-加载缺少错误处理.md`

4. `HomeRailProfilePanel.vue` + `homeShellMenu.service.ts`

- 问题：个人中心金刚区复用 drawer routeUrl，来源一直写成 `shell-drawer`。
- 影响：入口来源标识失真，后续统计、提醒消费、问题追踪会混线。
- 相关摘录：
  - `04-HomeRailProfilePanel-金刚区来源失真.md`
  - `05-homeShellMenu-service-入口来源定义.md`

5. `HomeRailProfilePanel.vue`

- 问题：内容链加载缺少 `try/catch`。
- 影响：个人中心页内容服务失败时会出现未处理 rejection。
- 相关摘录：`06-HomeRailProfilePanel-加载缺少错误处理.md`

6. `homeRailHome.model.ts` + `homeRailHomeContent.service.ts` + `HomeRailHomePanel.vue`

- 问题：banner 已建立 `focalPoint` 字段并从内容链映射出来，但页面未消费。
- 影响：后端后续即使下发焦点真值，首页 banner 仍只按默认 cover 裁切。
- 相关摘录：
  - `07-homeRailHome-model-banner-focalPoint.md`
  - `08-homeRailHomeContent-service-banner-focalPoint映射.md`
  - `09-HomeRailHomePanel-banner-未消费focalPoint.md`
