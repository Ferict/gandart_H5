# 批次396-活动页抽奖卡礼物图标与同构样式收口

## 本批目标

- 将抽奖活动卡完全收口到与资产合成相同的深色卡语法。
- 抽奖卡标签保持 `DRAW`。
- 去掉原本的英文副标题。
- 图标改为礼物语义，并放在标题上方。
- 图标视觉权重下调，避免抢过标题。

## 执行范围

- `src/pages/home/components/HomeRailActivityPanel.vue`
- `src/models/aetherIcon.model.ts`
- `src/utils/aetherIcon.registry.ts`

## 实际修改

1. 抽奖活动卡从浅色卡改回深色卡层，直接复用 `home-activity-entry-layer-dark`。
2. `DRAW` 标签改为与 `MERGE` 同口径，使用默认深色卡标签样式，不再走 `is-strong` 特化。
3. 标题保留置底，原英文副标题移除。
4. 图标改为 `gift`，置于标题上方，尺寸 `16px`，颜色改淡。
5. `gift` 已加入统一图标适配层：
   - H5 走 Lucide `Gift`
   - APP iconfont 当前无 gift glyph，临时回退到 `package` 近似礼盒形态，并在 registry 内留注释。

## 验证

- `npm run build:app`
- `npm run build:app-harmony`
