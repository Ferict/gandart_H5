# 批次386-首页顶部导航投影修复与市场卡圆角

## 目标

- 将首页顶部导航滚动态从“几乎不可见的均匀外发阴影”改为明确的向下投影。
- 给首页市场藏品卡补齐统一圆角，收口卡面与命中层的边界。

## 修改范围

- `src/pages/home/components/HomeRailTopbar.vue`
- `src/pages/home/components/HomeRailHomePanel.vue`

## 实际修改

### 1. 首页顶部导航滚动态改为向下投影

- `HomeRailTopbar.vue`
- 保留 `is-scrolled` 状态链，不改滚动状态来源。
- 将原有 `0 0 16px rgba(...)` 改为带 `y` 偏移的向下投影。
- 新增 `::after` 渐隐投射层，让顶栏阴影真正落到页面内容上。
- 同步补一点滚动态表面分离：背景加深、底边线显性化。

### 2. 首页市场藏品卡补圆角

- `HomeRailHomePanel.vue`
- 市场卡命中层、卡壳统一从 `12px` 收到 `16px`。
- 市场卡左侧 `4px` 青线同步补左侧圆角。
- 市场卡图片区改成顶部双圆角，和卡壳边界一致。

## 边界

- 不改顶部导航滚动触发逻辑。
- 不改 `HomeShellTrackStage.vue` 滚动容器实现。
- 不改首发卡、活动页卡片、个人中心卡片圆角。
- 不改首页其它阴影、hover 或按压反馈。

## 验证

- `npm run type-check`
- `npm run build:h5`
- `npm run build:app`
- `npm run build:app-harmony`
