# 批次360 首页 4/8px 容器修复

## 本批目标

- 按首页 `4/8px` 倍数审查结果，修复非字号类违规值。
- 本批不改字号本身，只修：
  - icon 尺寸
  - line-height
  - padding / gap
  - radius
  - blur / shadow
  - 装饰点尺寸

## 修改范围

- `src/pages/home/components/HomeRailActivityPanel.vue`
- `src/pages/home/components/HomeRailProfilePanel.vue`
- `src/pages/home/components/HomeShellDrawer.vue`
- `src/pages/home/components/HomeShellNavRail.vue`
- `src/pages/home/components/HomeRailTopbar.vue`

## 实际收口

- `18 / 22` 图标尺寸统一收口到 `20 / 24`
- `6 / 10` 间距统一收口到 `8 / 12`
- `14 / 18` 行高统一收口到 `16 / 20`
- `10 / 14 / 18` 圆角与阴影模糊统一收口到 `12 / 16 / 20`
- `6x6` 装饰点统一收口到 `8x8`

## 结果

- 首页当前剩余的 `4/8px` 倍数例外，主要只剩字号值：
  - `10 / 11 / 13 / 14 / 15`
- 这些值按本轮口径允许保留，后续不作为容器规范违规处理。
