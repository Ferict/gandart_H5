## 1. 目录定位

`留档/` 的目标固定为：

1. 让计划、审查、交付分层明确
2. 让关键行为可追溯、可反查
3. 让历史档案并入统一历史体系

`留档/` 不是：

1. 现行规则正文目录
2. 现行口径目录
3. 垃圾桶
