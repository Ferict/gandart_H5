# 新藏品
> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录用于承载用户提供或人工筛选的藏品候选素材原图，是“候选原图层”，不是运行时静态资源入口。

## 1. 目录职责

本目录固定负责：

1. 保存候选原图与用户提供的原始素材批次。
2. 作为运行时资源替换前的来源层，保留原始命名与未归一化图片。
3. 为首页、市场、个人页等 mock 资源替换提供可追溯的候选素材源。

## 2. 当前使用规则

1. 本目录只承接候选原图，不承接运行时代码直接消费的正式静态资源。
2. 被正式采用的运行时图片，应进入 `src/static/` 或正式资源链，而不是继续直接从本目录读取。
3. 本目录允许保留长文件名原图，不在当前批次强制重命名。
4. 一批素材完成筛选并稳定进入正式资源链后，可在后续批次决定是否归档或清退。

## 3. 常见内容

1. 用户提供的 `.png` 候选原图。
2. 曾用于首页与市场 mock 资源替换的原始素材批次。

## 4. 不应放入的内容

1. 运行时代码直接引用的正式静态资源。
2. `src/mocks/` 结构化数据文件。
3. 审查、计划、台账或对话记录。
4. 与藏品候选素材无关的临时文件。

## 5. 当前阶段结论

1. 经 `P5` 去留审查与 `P7.2` 素材分层审查，本目录当前继续保留为现行候选素材源目录，不进入本轮 `垃圾桶/`。
2. 本目录后续只有在对应素材已全部失效、且完成新资源链替换后，才应进入下一轮归档或清退审查。
