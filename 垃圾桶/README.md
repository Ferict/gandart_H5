## 1. 目录职责

本目录只用于承接当前仓内已经退出现行路径、但不适合直接物理删除的内容。

1. 已废弃的旧规则、旧模板、旧入口。
2. 已确认退出当前路径的历史壳、冻结快照和退役正文。

当前 Git 只跟踪本 `README.md`。

3. 不从现行规则、现行接口说明或现行源码回链到 `垃圾桶/` 内部对象。
4. 如需追溯历史，应优先查看 [留档/历史记录](/H:/工作/天异/uniapp+vue新架构/留档/历史记录/README.md) 或对应正式交付记录。
