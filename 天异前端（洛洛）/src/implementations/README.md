# src/implementations

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录承载内容接入链中的具体实现层，属于 `src` 长期蓝图中的内容接入链。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“内容接入链”。
2. 本目录只承接 provider 的具体实现，不承接页面私有业务逻辑。

## 目录职责

1. 存放 `http / mock` 等 provider 的具体实现。
2. 为 `ports` 提供具体可运行的接入实现。
3. 承接联调入口、mock 实现和未来真实后端 provider 的具体代码。

## 当前定位

1. 本目录只承接内容接入链相关新增。
2. `http` 和 `mock` 实现当前仍需并存，但必须受统一 provider 选择机制约束。
3. 后续若接入链做并轨治理，本目录属于并轨候选之一。

## 不应放入的内容

1. 页面私有 runtime。
2. 共享 UI、共享 store 或目录说明文档之外的历史档案。
3. 与 provider 实现无关的工具拼装逻辑。
