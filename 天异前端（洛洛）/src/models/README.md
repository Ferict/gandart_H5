# src/models

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-20
> 说明：本目录用于承载跨页面或跨业务共享的前端模型、schema 与领域对象，当前已按职责簇分组。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“稳定锚点”。
2. 本目录承担共享前端模型层，不承担 DTO 契约层和 provider 实现层。

## 目录职责

1. 放置共享模型定义。
2. 放置跨页面共享前端领域模型；页面私有 query schema 应靠近页面 runtime。
3. 通过职责分组降低模型文件的人工检索成本。

## 当前结构

1. `content/`
   - 内容缓存与资源详情模型。
2. `home-rail/`
   - 首页三条 rail 的共享模型。
3. `home-shell/`
   - 首页壳层与菜单模型。
4. `profile-asset-detail/`
   - 详情页模型。
5. `ui/`
   - UI 级共享模型。

说明：

1. `route-query/` 已随独立占位入口拆线退出共享模型层。
2. 藏品详情和统一建设中页的 query 解析已分别收回页面附近。

## 不应放入的内容

1. 接口 DTO 契约。
2. 具体 provider 实现。
3. 页面模板代码。
4. 仅服务于单页面的临时类型别名。
5. 页面私有 route query 模型。
