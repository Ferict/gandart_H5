# src/stores

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录承载应用级共享 store，属于 `src` 长期蓝图中的稳定共享层。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“稳定共享层”。
2. 本目录只承接应用级或跨页共享状态，不承接页面局部状态。

## 目录职责

1. 存放 Pinia 或等价共享状态 store。
2. 承接应用级身份、服务中枢和跨页共享状态。
3. 为页面层和共享运行时提供集中状态入口。

## 当前定位

1. 当前目录保持小而稳，不以扩容 store 数量为目标。
2. 页面局部状态应继续留在页面就近层。
3. 只有明确存在跨页共享价值时，才新增 store。

## 不应放入的内容

1. 页面局部状态或一次性中间状态。
2. 接口实现、mock 数据和静态资源说明。
3. 只为规避参数传递而创建的伪共享 store。
