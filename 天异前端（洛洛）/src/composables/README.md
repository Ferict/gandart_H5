# src/composables

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录承载跨页共享 composable，属于 `src` 长期蓝图中的稳定共享层。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“稳定共享层”。
2. 本目录只承接真正跨页共享的前端组合逻辑，不承接页面私有 runtime。

## 目录职责

1. 存放多个页面或共享组件可复用的组合逻辑。
2. 承载布局、滚动反馈、图片状态、轨道指示等跨页共享运行时。
3. 为 `pages/*` 下的 page-local runtime 提供可复用的共享基础能力。

## 当前定位

1. 页面私有 composable 不再放入本目录。
2. 页面专属运行时应优先靠近对应 `pages/*` 维护。
3. 跨业务重型 runtime 优先进入 `services/*` 或保留在页面就近层，不回流到本目录。

## 不应放入的内容

1. 单页面专属 composable。
2. 接口实现、mock 数据或 provider 适配逻辑。
3. 只为局部模板拆分而生成的 page-local helper。
