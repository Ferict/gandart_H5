# src/mocks

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录用于承载结构化 mock 数据源与 mock 数据库结构，是“mock 数据引用层”，不是图片原图目录。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“稳定锚点”。
2. 本目录只承接 mock 与开发数据支撑层，不承接正式运行时静态资源层。

## 1. 目录职责

1. 放置 mock 内容数据库与场景数据。
2. 作为 `content.mock.ts` 等实现体可读取的结构化静态数据源。
3. 通过数据字段引用正式运行时静态资源路径。

## 2. 当前内容

1. `content-db/`
2. `updating-db/`

## 3. 当前使用规则

1. 本目录只承接结构化数据，不承接原图素材和正式静态资源文件本体。
2. 如需引用图片，应引用 `src/static/` 中的正式运行时资源路径，而不是复制图片进来。
3. 本目录中的说明文件只说明数据结构与边界，不承担素材来源登记职责。

## 4. 不应放入的内容

1. 正式接口实现。
2. 跨页面运行时 service。
3. 共享 UI 组件。
4. 候选原图或运行时静态图片文件。
