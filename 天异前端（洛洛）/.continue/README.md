# .continue

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-13
> 说明：本目录用于承载本仓接入 Continue 类本地辅助工具时所需的检查配置与辅助文件，不作为现行规则真值源，也不承载业务源码。

## 1. 目录职责

本目录当前用于：

1. 保存 Continue 工具相关的本地检查目录与辅助配置。
2. 为本地开发辅助流程提供非业务配置落点。

## 2. 使用边界

1. 本目录不是业务代码目录。
2. 本目录不是正式规则目录。
3. 本目录不是历史记录目录。

## 3. 不应放入的内容

以下内容不应写入本目录：

1. 页面、组件、service、composable 等业务源码
2. 现行规则正文
3. 正式接口契约
4. 审查、交付、历史记录正文
