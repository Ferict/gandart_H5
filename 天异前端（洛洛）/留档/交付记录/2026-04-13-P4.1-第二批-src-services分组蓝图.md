## 4. 执行边界

下一批物理迁移时只做：

1. 文件移动。
2. 导入路径回写。
3. 新子目录 `README.md` 补齐。
4. 父目录 [src/services/README.md](/H:/工作/天异/uniapp+vue新架构/src/services/README.md) 口径更新。

下一批不做：

1. service 改名。
2. service 职责重写。
3. service API 改签名。
4. `src/services/` 之外的结构大重构。

## 5. 验证要求

物理迁移完成后必须执行：

1. `git diff --check`
2. `npm run type-check`
3. `npm run knip`
4. `npm run test:unit`
