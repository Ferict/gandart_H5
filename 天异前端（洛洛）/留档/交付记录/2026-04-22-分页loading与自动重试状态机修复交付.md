## 1. 做了什么

1. 新增 `useHomeRailPaginationLoadingChain`，固定分页 loading 状态、错误重试、无进展重试、请求超时、冷却、单飞和版本守卫。
2. 首页藏品 load more 从 `boolean` 返回改为结构化结果，能区分 `appended / no-progress / endline / error / stale`。
3. 个人藏品 load more 同步改为结构化结果，并接入同一分页状态机。
4. 公告非首屏刷新接入自动重试链，底部 footer 只在真实 pagination loading、错误或 endline 时显示。
5. 首页藏品追加进入动画在没有 mounted queue 时立即释放 `entering` 与高度锁，避免后续分页触发被卡住。
6. 三处 footer 稳定态不再默认显示 loading。
7. 补充状态机、首页藏品远端分页、首页 load more、展示层和公告远端状态相关单测。
