## 4. 结构问题

### 4.1 `HomeRailHomePanel.vue`

当前文件同时承载以下职责：

1. 首页 scene shell 与内容 hydrate
2. 公告、banner、featured、market 的内容分发
3. 市场列表 query、remote list、result window
4. 页面 presentation runtime 与视觉刷新 key
5. placeholder icon、focal point、底部 footer 等展示计算
6. 点击导航和 target URL 构建
7. 首屏 hydrate、激活检查、可见性检查、watcher 协调

判断：

1. 这不是单纯的“模板大”，而是页面级 orchestration 仍然堆在组件里。
2. 当前虽然已经消费多个 composable，但组合层仍然过厚，阅读时需要在一个文件里跨越过多主题。

### 4.2 `HomeRailProfilePanel.vue`

当前文件同时承载以下职责：

1. 分类与子分类 query 状态
2. 个人页 scene adapter 与 runtime
3. 资产列表 remote list、reload、result window
4. 可见资产数量、load-more、observer、prefetch
5. quick action、地址展示、详情跳转
6. 首屏 hydrate、激活检查、可见更新检查、watcher 协调

判断：

1. 个人页组件的核心问题与首页市场 rail 相同，也是 orchestration 与运行时状态过厚。
2. 该文件对人工维护最不友好的部分不是模板，而是脚本段中多条状态链并列存在。

### 4.3 `HomeRailActivityPanel.vue`

当前文件同时承载以下职责：

1. 活动公告 query、remote list、result window
2. notice read bridge 与 visual reveal
3. 空态、底部 footer、首屏失败与重试计算
4. refresh、activation、visible update 检查
5. 多组 watcher 的协调

判断：

1. 相比首页市场 rail 和个人页 rail，活动页巨石问题较轻，但仍然明显。
2. 它更适合作为后续收尾批次，而不是第一刀。

### 4.4 `HomeShellTrackStage.vue`

当前文件主要负责：

1. track stage 的装配
2. 三个 rail panel 的挂载与 refresh handle 连接
3. runtime insets、tabbar、drawer 事件桥接

判断：

1. 当前文件虽然 import 较多，但职责相对集中。
2. 它更像“首页轨道壳”，暂不建议优先拆。

## 5. 当前最值得优先治理的问题

首页当前最影响人工维护的问题已经明确收敛为：

1. [HomeRailHomePanel.vue](/H:/工作/天异/uniapp+vue新架构/src/pages/home/components/HomeRailHomePanel.vue) 的脚本组合层过厚
2. [HomeRailProfilePanel.vue](/H:/工作/天异/uniapp+vue新架构/src/pages/home/components/HomeRailProfilePanel.vue) 的脚本组合层过厚
3. [HomeRailActivityPanel.vue](/H:/工作/天异/uniapp+vue新架构/src/pages/home/components/HomeRailActivityPanel.vue) 的脚本组合层中度过厚

这三个问题都属于：

1. 组件内部职责治理问题
2. 不是目录平铺问题
3. 不适合再用“继续拆更多微文件”来机械解决

## 6. 拆分边界建议

下一阶段对首页巨型组件的治理，应遵守以下边界：

1. 先拆脚本 orchestration，不先拆模板组件树。
2. 优先抽“成组运行时职责”，不抽零碎小 helper。
3. 单次只处理一个 rail panel，避免三页一起动。
4. 新增 composable 应优先放在：
   - [src/pages/home/composables/home](/H:/工作/天异/uniapp+vue新架构/src/pages/home/composables/home/README.md)
   - [src/pages/home/composables/profile](/H:/工作/天异/uniapp+vue新架构/src/pages/home/composables/profile/README.md)
   - [src/pages/home/composables/activity](/H:/工作/天异/uniapp+vue新架构/src/pages/home/composables/activity/README.md)
5. 不允许为了降行数继续制造无意义微文件。
