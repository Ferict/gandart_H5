# 计划记录

> 文档类型：目录入口
> 状态：active
> 更新时间：2026-04-15
> 说明：本目录用于承载正式计划、主线计划、分批计划和执行状态回填入口。

## 1. 目录职责

1. 存放正式计划
2. 存放主线计划
3. 存放分批计划
4. 存放已执行计划的状态回填入口

## 2. 使用规则

1. 新计划只进入 `留档/计划记录/`
2. 计划开始执行后，必须在原计划文中回填执行状态与执行索引
3. 计划完成后的结果进入 `留档/交付记录/`
4. 旧路径中的同名计划文只保留为冻结入口
