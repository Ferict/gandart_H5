# 批次381-首页标题系统同步到活动页与个人中心

## 范围

- `src/pages/home/components/HomeRailActivityPanel.vue`
- `src/pages/home/components/HomeRailProfilePanel.vue`

## 目标

- 将活动页与个人中心中和首页同构的标题区、section head、动作位，收口到首页现行标题系统口径。
- 仅调整标题行容器、英文副标题基线、右侧动作位图标尺寸与间距。
- 不改卡片内容、不改数据链、不改交互语义。

## 实际修改

1. 活动页

- `home-activity-header` 从 `align-items: center` 改为 `align-items: flex-end`，补 `gap: 8px`
- `home-activity-section-head` 从 `align-items: center` 改为 `align-items: flex-end`，补 `gap: 8px`
- `EVENTS / UPDATES` 英文副标题补 `vertical-align: baseline`

2. 个人中心

- `home-profile-header` 从 `align-items: center` 改为 `align-items: flex-end`，`gap: 12px -> 8px`
- `USER_SYNC / ASSETS` 英文副标题补 `vertical-align: baseline`
- 头部动作位和“我的藏品”搜索动作位改成首页市场动作位口径：
  - 图标 `20 / 1.8 -> 16 / 2`
  - 动作区间距 `8 -> 12`
  - 容器补 `border-radius: 999px`、`flex: 0 0 auto`、`overflow: visible`

## 边界

- 不扩到首页组件
- 不改活动页入口卡和公告卡
- 不改个人中心身份卡、摘要卡、资产卡数据和样式
- 不改服务层、model、mock 数据
