### 批次336：活动页与个人中心内容链错误处理收口

问题：

- `HomeRailActivityPanel.vue` 与 `HomeRailProfilePanel.vue` 在 `onMounted` 直接 await 内容服务，失败时会出现未处理 rejection
- 收尾同步逻辑没有统一落在 finally

修复：

- 活动页改为 `try/catch/finally`
- 个人中心改为 `try/catch/finally`
- 失败时回落到对应 shell 内容
- 个人中心提醒预热改为显式 `.catch`

结果：

- 活动页、个人中心页内容加载失败时不再抛未处理 rejection
- 现有 `syncActiveTag()`、`syncProfileFilters()` 收尾逻辑仍然保留

代码前后对照：

修复前：

```ts
onMounted(async () => {
  activityContent.value = await resolveHomeRailActivityContent()
  syncActiveTag()
})
```

```ts
onMounted(async () => {
  void ensureHomeShellMenuReminderHydrated()
  content.value = await resolveHomeRailProfileContent()
  syncProfileFilters()
})
```

修复后：

```ts
onMounted(async () => {
  try {
    activityContent.value = await resolveHomeRailActivityContent()
  } catch (error) {
    console.error('[homeRail] failed to resolve activity scene content', error)
    activityContent.value = createHomeRailActivityContentShell()
  } finally {
    syncActiveTag()
  }
})
```

```ts
onMounted(async () => {
  void ensureHomeShellMenuReminderHydrated().catch((error) => {
    console.error('[homeRail] failed to hydrate service reminders', error)
  })

  try {
    content.value = await resolveHomeRailProfileContent()
  } catch (error) {
    console.error('[homeRail] failed to resolve profile scene content', error)
    content.value = createHomeRailProfileContentShell()
  } finally {
    syncProfileFilters()
  }
})
```

### 批次337：banner 消费 focalPoint

问题：

- `homeRailHome.model.ts` 与 `homeRailHomeContent.service.ts` 已支持 `focalPoint`
- 页面层未消费，banner 仍只按默认中心裁切

修复：

- `HomeRailHomePanel.vue` 增加 `resolveBannerImageStyle`
- banner 图片节点按 `focalPoint` 绑定 `object-position`
- 无 `focalPoint` 时回退到 `50% 50%`

结果：

- banner 焦点字段已从 model/service 真正落到页面裁切位置
- 后端后续下发不同焦点时，首页页面层已可正确消费

代码前后对照：

修复前：

```vue
<image v-if="item.imageUrl" class="home-banner-bg" :src="item.imageUrl" mode="aspectFill" />
```

修复后：

```vue
<image
  v-if="item.imageUrl"
  class="home-banner-bg"
  :src="item.imageUrl"
  :style="resolveBannerImageStyle(item)"
  mode="aspectFill"
/>
```

```ts
const resolveBannerImageStyle = (item: HomeBannerItem): CSSProperties => {
  if (!item.focalPoint) {
    return { objectPosition: '50% 50%' }
  }
  return {
    objectPosition: `${item.focalPoint.x * 100}% ${item.focalPoint.y * 100}%`,
  }
}
```
