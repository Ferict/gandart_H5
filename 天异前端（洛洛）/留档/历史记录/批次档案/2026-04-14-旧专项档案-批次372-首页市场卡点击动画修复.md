# 批次372-首页市场卡点击动画修复

## 目标

- 修复首页市场卡点按时出现直角方框的问题。
- 给市场卡补上明确的按压反馈，而不是只出现一层生硬覆盖。

## 实际修改

- 文件：`src/pages/home/components/HomeRailHomePanel.vue`
- 调整项：
  1. `home-market-card-entry` 增加 `border-radius: 12px`，使 `HomeInteractiveTarget` 的 block hit 层继承正确圆角。
  2. `home-market-card-entry.is-entry-active` 增加整体轻微缩放。
  3. `home-market-card-visual` 在按压时增加轻微下沉与阴影变化。
  4. `home-market-card-copy` 在按压时同步轻微下移，保证图文反馈一致。

## 结果

- 市场卡点击/触摸时不再出现直角方框。
- 市场卡现在具备卡面级按压反馈，而不是只有生硬的命中层覆盖。

- `npm run type-check` 通过
- `npm run build:h5` 通过
- `npm run build:app` 通过
- `npm run build:app-harmony` 通过
