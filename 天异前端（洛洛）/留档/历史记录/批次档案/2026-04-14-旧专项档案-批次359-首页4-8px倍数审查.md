# 批次359 首页 4/8px 倍数审查

## 审查边界

- 只审首页 `src/pages/home/**`
- 只审实际 UI 尺寸、字号、图标尺寸、间距、圆角、阴影等可见设计值
- 不纳入以下例外：
  - `1px / 2px` 线框
  - `999px` 胶囊圆角
  - 壳层阈值与基准宽：`375 / 430 / 522 / 768`

## 审查结论

- 首页当前仍存在多处不符合 `4/8px` 倍数口径的值。
- 主要集中在：
  - 字号与行高
  - 局部 icon 尺寸
  - 局部 gap / padding
  - 局部圆角与阴影模糊半径
