# 审核 - 新仓库启动与 uni-app + Vue 规范迁移

- 根 Git 已转移到 `../旧架构文件`
- `uniapp+vue 新架构` 已初始化为新 Git 仓库
- 旧仓规范和管理文件已复制进新仓，但内容仍主要指向旧 React/Vite 架构

3. 新仓必须重新建立：

   - 项目台账
4. 对话台账规则应保留，但应补录本轮迁移对话。

## 3. 本批不处理项

- 不创建 uni-app 业务源码
- 不搭建页面
- 不迁移旧仓历史审核堆栈正文
- 不修改旧仓任何文件

- 若继续沿用旧规范，会把新仓的技术链路定义错。
- 若新仓不先建立管理边界，后续再写代码会继续混入旧仓口径。
