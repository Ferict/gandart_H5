# 批次384-首页顶部导航滚动投影复查

## 审查目标

- 复查首页顶部导航栏在滚动后是否已经实现对页面的投影效果。
- 判断是未实现，还是实现了但效果过弱/不易感知。

## 审查范围

- `src/pages/home/components/HomeRailTopbar.vue`
- `src/pages/home/components/HomeShellTrackStage.vue`

## 结论

1. 这条不是没做。

- `HomeRailTopbar.vue` 已有滚动态样式：
  - `.home-topbar.is-scrolled { box-shadow: 0 0 16px rgba(15, 23, 42, 0.06); }`
- `HomeShellTrackStage.vue` 已有滚动状态传递：
  - `handleTrackScroll()` 中 `scrollTop > 0` 时把 `homeTrackScrolled` 置为 `true`
  - `HomeRailTopbar` 通过 `:is-scrolled="homeTrackScrolled"` 接收该状态

2. 当前更准确的问题是“效果没展现清楚”。

- 投影值太轻：`0 0 16px rgba(15, 23, 42, 0.06)`
- 阴影是均匀四周发散，不是明显的向下投影
- 顶栏背景色和页面背景色都接近 `#fafafa`，对比不足
- 顶栏本身是绝对定位悬浮，用户预期更像“滚动后产生底部压感”，但现实现更像轻微雾化边缘

3. 当前链路边界

- 这条投影只命中首页顶部栏
- `handleTrackScroll()` 开头有 `if (!isHomePage.value) return`
- 活动页、个人中心当前不参与这条滚动投影链

## 审查结论一句话

- 已做，但做得太轻，而且方向不对；用户想要的是“向页面产生投影”，现实现是“很弱的均匀外发阴影”。
