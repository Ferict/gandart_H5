# 批次375-首页卡面阴影复查

## 目标

- 复核用户指出的“阴影还在”到底来自按压态，还是来自卡片常驻视觉阴影。

## 审查范围

- `src/pages/home/components/HomeRailHomePanel.vue`

## 审查结果

1. 首页市场卡按压态附加阴影已删除。

- `home-market-card-entry.is-entry-active .home-market-card-visual` 当前只剩 `transform`，没有 `box-shadow`。

2. 用户截图里看到的阴影来自市场卡静态卡面阴影，而不是按压态阴影。

- `home-market-card-visual` 仍保留：
  - `box-shadow: var(--aether-shadow-soft, 0 0 24px rgba(15, 23, 42, 0.06));`
- 这条是常驻视觉样式，未在上一批删除。

3. 首页首发右侧视觉卡也仍保留同口径的静态卡面阴影。

- `home-featured-visual` 仍保留：
  - `box-shadow: var(--aether-shadow-soft, 0 0 24px rgba(15, 23, 42, 0.06));`

## 结论

- 当前实现与“不要阴影”的口径仍不一致。
- 不一致点不在按压态，而在卡片视觉层的常驻阴影。
- 若要彻底收口，下一批应删除首页卡面静态阴影，而不是继续改按压态。
