# 批次358 首页藏品占位图标放大

## 本批目标

- 放大首页藏品卡片空状态/未加载状态的占位图标。
- 不改图片链、不改卡片结构、不改占位图标种类。

## 修改范围

- `src/pages/home/components/HomeRailHomePanel.vue`

## 实际修改

- 首发藏品卡占位图标：
  - `40 -> 24`
- 藏品市场卡占位图标：
  - `30 -> 40`

## 结果

- 占位图标在空状态下更易识别。
- 一旦图片资源存在，仍由图片分支优先显示，不与真实藏品图片冲突。

## 验证

- `npm run type-check`
- `npm run build:h5`
- `npm run build:app`
- `npm run build:app-harmony`
