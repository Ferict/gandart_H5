# 批次373-首页公告点击态与首发信息栏跳转

## 目标

- 修复首页公告条点击层露直角 hit 层的问题。
- 让首发藏品左侧信息栏与右侧视觉卡统一打开同一详情页。

## 修改文件

- `src/pages/home/components/HomeRailHomePanel.vue`

## 实际修改

1. 公告条点击层收口

- 给 `home-notice-hit` 根节点补齐圆角与 overflow。
- 给 `home-notice-bar` 补过渡与按压态联动，避免 block 命中层露出直角方框。

2. 首发信息栏接入点击

- 将左侧 `home-featured-info-card` 从普通 `view` 改为 `HomeInteractiveTarget`。
- 绑定与右侧视觉卡一致的 `handleCollectionQuickEntryClick()`。
- 补齐按压态下沉与亮度联动，保持与首页市场卡同口径。

## 边界

- 只改首页公告条与首发信息栏。
- 不扩到活动页公告卡。
- 不改首页数据链与详情页。

## 验证

- `npm run type-check`
- `npm run build:h5`
- `npm run build:app`
- `npm run build:app-harmony`
