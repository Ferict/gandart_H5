# src/utils

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录承载跨页共享、语义较轻的工具函数与注册表，属于 `src` 长期蓝图中的稳定共享层。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“稳定共享层”。
2. 本目录只承接轻量共享工具，不承接重型运行时或页面私有 helper。

## 目录职责

1. 存放跨页复用且业务语义较轻的工具函数。
2. 承载图标注册表、时间格式化、路由 query 工具、安全日志、结果区共享展示 util。
3. 为共享层和 page-local runtime 提供基础级工具支持。

## 当前定位

1. 本目录已经开始承接跨 rail 共享 util，例如结果区展示和 template-ref 解析。
2. 页面私有 helper 仍应优先靠近 `pages/*`。
3. 当工具开始演化为重型 runtime 或业务 service 时，应迁出本目录。

## 不应放入的内容

1. 跨业务重型 service。
2. 页面私有 helper 或模板拆分辅助。
3. 正式接口 DTO 契约和 provider 实现。
