# src/static

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录用于承载运行时真正消费的正式静态资源，是“运行时静态资源层”。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“稳定锚点”。
2. 本目录只承接正式运行时静态资源，不承接候选原图和结构化 mock 数据。

## 1. 目录职责

1. 放置字体、logo、图标字体与各页面正式运行时静态资源。
2. 承载被代码、mock 数据或页面模板真实消费的静态素材。
3. 对被正式采用的候选素材提供归一化命名后的运行时落点。

## 2. 当前内容

1. `fonts/`
2. `home/`
3. `iconfont/`
4. `logo/`

## 3. 当前使用规则

1. 本目录只承接正式运行时静态资源，不承接候选原图。
2. 被正式采用的候选图片，应先归一化后进入本目录，再由代码或 mock 数据引用。
3. 本目录允许保留少量目录级来源登记或说明文件，但不承接流程文档与历史档案。

## 4. 不应放入的内容

1. 仅供筛选的原始候选素材。
2. 审查、计划、台账记录。
3. 运行时代码逻辑。
4. 历史回档材料与已失效素材批次。
