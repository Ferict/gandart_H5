# src/contracts

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-14
> 说明：本目录承载内容接入链中的正式 DTO 契约层，属于 `src` 长期蓝图中的内容接入链。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“内容接入链”。
2. 本目录只承接正式接口 DTO、schema 和与接入链直接相关的契约结构。

## 目录职责

1. 存放内容域正式 DTO 契约定义。
2. 作为 `ports / implementations / adapters` 的上游数据结构真值源之一。
3. 保持接口字段、分页结构和资源形态的稳定前端镜像。

## 当前定位

1. 本目录只承接内容接入链相关新增。
2. 当前仍以内容域契约为核心，不扩展无关业务领域。
3. 后续若接入链做并轨治理，本目录属于并轨候选之一。

## 不应放入的内容

1. provider 具体实现代码。
2. 页面私有 model 或展示态派生。
3. 与正式接口无关的临时类型占位。
