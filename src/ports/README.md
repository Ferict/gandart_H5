# src/ports

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-20
> 说明：本目录承载内容接入链中的抽象端口层，属于 `src` 长期蓝图中的内容接入链。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“内容接入链”。
2. 本目录只承接抽象端口和上层可消费的 provider 能力边界。

## 目录职责

1. 存放内容域所需的抽象端口定义。
2. 约束 `implementations` 和上层 `services` 之间的能力边界。
3. 为 provider 切换、mock 对接和真实联调提供抽象接口层。

## 当前定位

1. 本目录只承接内容接入链相关新增。
2. `pageContent.port.ts` 已随独立占位入口拆线退出，不再保留 notice/updating 占位内容端口。
3. 后续若接入链做并轨治理，本目录属于并轨候选之一。

## 不应放入的内容

1. provider 具体实现代码。
2. 页面私有临时接口或局部 helper。
3. 与接入链无关的共享模型和展示态派生。
