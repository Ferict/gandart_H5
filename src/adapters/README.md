# src/adapters

> 文档类型：目录说明
> 状态：active
> 更新时间：2026-04-20
> 说明：本目录承载内容接入链中的适配层，属于 `src` 长期蓝图中的内容接入链。

## 蓝图定位

1. 本目录在 `P7.3` 长期蓝图中属于“内容接入链”。
2. 本目录负责把上游接口转换为前端可消费形态，不直接承接页面私有逻辑。

## 目录职责

1. 存放内容接入链中的适配层实现。
2. 处理正式内容接入链需要的转换。
3. 为 `services/content/*` 和上层消费方提供稳定适配输出。

## 当前定位

1. 本目录只承接内容接入链相关新增。
2. `pageContent.mock.adapter.ts` 已随独立占位入口拆线退出，不再保留 notice/updating 占位 preset mock。
3. 后续若接入链做并轨治理，本目录属于并轨候选之一。

## 不应放入的内容

1. 页面私有运行时。
2. 共享 UI 组件或共享 store。
3. 与内容接入链无关的新业务逻辑。
