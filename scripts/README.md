# scripts

> 文档类型：目录入口
> 状态：active
> 更新时间：2026-04-13
> 说明：本目录承载当前仓库的维护脚本、质量工具脚本和最小联调验证脚本。

## 1. 目录职责

本目录承载：

1. 质量工具脚本
2. 联调验证脚本
3. 环境初始化脚本

本目录不承载：

1. 业务运行时代码
2. 页面逻辑
3. 正式规则正文

## 2. 当前结构

1. `quality/`
   - 质量门禁、检查和自动化相关脚本
2. `check-content-http-link.mjs`
   - 直接读取单一 contract 真值源的内容域 5 接口联调验证矩阵
3. `check-content-http-link.shared.mjs`
   - 为联调脚本提供 contract 读取、验证矩阵组装与可测试的共享逻辑
4. `setup-quality-tools.ps1`
   - Windows 质量工具初始化入口

## 3. 不应放入的内容

1. 一次性人工临时命令记录
2. 页面专属工具逻辑
3. 需要长期保留的审查与交付记录
