## 6. 已冻结的使用规则

后续 `P11.12` 必须按以下规则使用本基线：

1. `confirmed` 只表示旧源码证据支撑的 transport 层可用，不表示 UI 可以直接吃 DTO。
2. `exported-unused` 只表示 wrapper 已导出但旧前端稳定使用证据不足，必须标注“仅候选，不可直接接 UI”。
3. `unresolved.md` 表示缺证据、缺响应样例、缺稳定字段消费或其他接受风险，必须继续待确认。
4. `contracts/inferred/*` 只作为缺失导出或推断线索，不得作为正式 API path。
5. `confidence.md` 中的 Level B 只能辅助差异矩阵，不得升级成已签收后端字段。
6. `FrontendUnknown` 响应必须由 adapter 或真实响应样例进一步收窄。
7. 页面与组件不得直接 import 本目录契约。
