# gandart_H5

Clean H5 source snapshot for frontend development, integration, and CI.

This repository keeps:

1. Runtime source code and static assets in `src/`
2. Tests in `tests/`
3. Engineering scripts in `scripts/`
4. GitHub Actions under `.github/`
5. Current frontend-consumed contract code under `src/contracts/`

This repository does not keep:

1. Internal rule documents
2. Archive and planning records
3. Source-material and handoff/reference directories
4. AI-eval and in-repo governance assets

Common commands:

```bash
npm install
npm run dev:h5
npm run quality:check
npm run test:e2e
```
